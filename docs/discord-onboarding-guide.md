# 陽だまりの庭 — Discord ピン留め用ガイド

以下をそのままDiscordに貼り付けてピン留めしてください。

---

## ① #✅ats_check-in チャンネル用（ピン留め）

```
🌸 陽だまりの庭へようこそ！

まず最初に **ウォレットの登録** をしてね。
下のコマンドをこのチャンネルに送るだけでOKだよ！
```
!登録
```
👆 送信すると登録リンクが届くよ。ボタンをクリックして完了！

━━━━━━━━━━━━━━━━━━━━━━
📌 登録の手順

① このチャンネルで `!登録` と送る
② ひまりちゃんからリンクが届く
③ ボタンをクリック → Discordでログイン
④ 自動でウォレットが作られて登録完了🎉
⑤ #📝今日の合言葉 チャンネルで合言葉を入力！

━━━━━━━━━━━━━━━━━━━━━━
❓ うまくいかないときは
・リンクの有効期限は **10分** だよ。切れたらもう一度 `!登録` してね
・登録済みの場合は「登録済みだよ」と返信が来るよ
```

## ② #📝今日の合言葉 チャンネル用（ピン留め）

```
🌱 合言葉の受け取り方

① 毎朝 **5:30〜** のスペースに参加
② このチャンネルに **今日の合言葉** を入力
③ ひまりちゃんから **3 ATSC** が届く🎉

━━━━━━━━━━━━━━━━━━━━━━
📌 注意

・1日1回まで（同じ日は2回受け取れないよ）
・ウォレット未登録の場合は #✅ats_check-in で `!登録` してね
・DawnSeed（参加証）が必要だよ。未取得の場合は管理人へ連絡を！

━━━━━━━━━━━━━━━━━━━━━━
🌳 ATSCの使い道

貯めたATSCはBioNFT（育成型NFT）の成長に使えるよ。
半年コツコツ続けると最終ステージ「大樹」になれるよ🌳
```

## ③ #✒️ひまりの記録帳 チャンネル用（ピン留め）

```
📊 ステータス確認・EXP→ATSC変換

▼ 自分のEXPとウォレットを確認
```
!ステータス
```

▼ 貯めたEXPをATSCに変換（10 EXP = 0.1 ATSC）
```
!convert
```

━━━━━━━━━━━━━━━━━━━━━━
🌸 EXPの貯め方

・☕陽だまりラウンジ・📝今日の合言葉でメッセージ → +1（1分に1回）
・🌸リアクションを押す → +1（1日5回まで）
・🌸リアクションをもらう → +2
```
